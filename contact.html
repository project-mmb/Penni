About This Project

Concept

A modern website for Penni, an AI-powered financial advisor, designed to demonstrate proficiency in creating professional, user-friendly web applications.

Design

Modern tech-centered design with purple, black, and white color scheme, featuring smooth animations and responsive layout for all devices.

Technology

Built with HTML5, CSS3, and JavaScript, incorporating modern web standards, responsive design, and interactive features.

Inspiration

Originally created for submission to a local bank's hackathon in Southern Africa, focusing on innovative financial technology solutions. Unfortunately I didn't manage to meet the submission deadlines so I decided to make it a personal project for my portfolio
